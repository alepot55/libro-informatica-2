specifica e quando emergono preferenze dell'utente. Vedi [[project-overview]].

Default ragionevoli adottati finché non specificato altrimenti:

- Markdown; code fence con linguaggio esplicitato; intestazioni gerarchiche.
- Terminologia tecnica in forma originale (es. `array`, `compiler`), spiegata
  alla prima occorrenza.

  includerli.
